## My version
I changed the colours and added the date time as
```
┌─[crud@localhost]─[~/Downloads/bash_dotfiles]-[Fri Aug 31 12:50:50]

```

## Installation
### Download and change folder
```
cd ~
git clone https://github.com/flipsidecreations/dotfiles.git

cd dotfiles
```
### Run intstall
```
./install.sh
```
